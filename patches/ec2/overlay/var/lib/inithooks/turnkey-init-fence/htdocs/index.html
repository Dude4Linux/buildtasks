Welcome to TurnKey! You need to initialize this system first before you can use it. To do that you'll need to log into the root account. The turnkey-init initialization program will start automatically:

How do I log into the root account?

You can authenticate in one of two ways:

  1. Without a password, using your SSH key-pair: If you've correctly configured an AWS SSH key-pair, you'll be able to log into the root account without having to enter a password using your SSH client (e.g., Putty on Windows).

  2. With a password: For security reasons the root password is initially set to a random value. To find the initial random root password, right click the instance in the AWS Management console, select "Get System Log":

    Then scroll down until you see LOGIN DETAILS in the log:
